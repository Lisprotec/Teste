# SCIE MAP Generator · GitHub Pages

Site estático para GitHub Pages que:

- recolhe os dados do caso;
- classifica a utilização-tipo e a categoria de risco;
- gera um dossiê base de medidas de autoproteção;
- exporta o relatório em PDF e JSON diretamente no browser.

## Como publicar no GitHub Pages

1. Cria um repositório no GitHub.
2. Carrega os ficheiros `index.html`, `styles.css`, `app.js` e `README.md`.
3. Em **Settings > Pages**, escolhe a branch principal e a pasta `/root`.
4. Guarda e aguarda a publicação.

## Observações importantes

- O motor implementa a classificação de risco das utilizações-tipo I a XII com base nos quadros do anexo III da Lei n.º 123/2019.
- As medidas de autoproteção seguem o quadro XXXIX e os artigos 199.º a 207.º da Portaria n.º 135/2020.
- Para UT XI e XII, a densidade de carga de incêndio modificada deve ser corretamente calculada e justificada em memória própria.
- O relatório gerado é uma base técnica. A validação legal e a submissão formal dependem da situação concreta e da intervenção de técnico habilitado quando exigível.
